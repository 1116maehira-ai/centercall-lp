11 — FAQ

よくいただく
ご質問。

本当に1日¥11,000で試せるんですか？
初回1回のみ、新規クライアント様向けの特別価格です。2日目以降は¥16,000/日となります。1日試して合わなければそこで終了、追加費用なし。
クレームが起きた時は誰が対応？
CenterCallが一次対応します。即時架電停止 → 録音内容を確認 → 24時間以内にクライアント様へ報告。
リスト（架電先）は自分で用意する必要が？
不要です。Google Maps API + Custom Search API + 独自リサーチでこちらが構築します。クライアント側のNG企業（競合・取引先）はお申し付けいただければ除外します。
録音データは誰が見られるんですか？
クライアント様とCenterCall担当者のみ。標準1ヶ月保管、契約終了から30日後に完全削除。録音延長オプション（¥5,000/月）で1年保管も可能。
アポが取れた後のクロージングはどうする？
標準では商談はクライアント様で実施。希望される場合はクロージング代行オプション（¥16,000/件 + 成約額の10%）で同席することも可能です。
